# Contributing

Contributions are welcome. Please open an issue first to discuss what you would like to change.

## Running locally

```bash
git clone https://github.com/Speed-boo3/soc-grc-project.git
cd soc-grc-project
pip install -r requirements.txt
```

## Project structure

- `soc/` — Security Operations Center tools
- `grc/` — Governance, Risk and Compliance tools
- `explain/` — Interactive learning site
- `scripts/` — Automated report generation
- `reports/` — Auto-generated compliance reports
